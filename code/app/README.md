# Веб-приложение к магистерской ВКР

Интерактивная демонстрация гибридного метода устранения помеховых
сигналов при измерении диаграмм обратного рассеяния в безэховых камерах
с неудовлетворительными безэховыми условиями.

## Локальный запуск

Из корня репозитория:

```
py -m pip install -r code/app/requirements.txt
py -m streamlit run code/app/app.py
```

Streamlit откроется по адресу `http://localhost:8501`.

## Структура

```
code/app/
├── app.py                  # главная страница
├── pages/
│   ├── 1_Анализ.py         # основной пайплайн обработки
│   └── 2_О_работе.py       # описание работы и авторов
├── core/
│   ├── synthetic.py        # генерация синтетических IQ-данных
│   └── processing.py       # пайплайн: ISAR, подавление помех, метрики
└── requirements.txt
```
